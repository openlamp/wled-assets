# Catalogue des animations d'effets (97 motions)

Chaque effet WLED est classé sur l'un de ces **97 motions** (mouvement). Les GIF s'animent
directement sur GitHub. Le nom = le `motion` ; en dessous, quelques effets qui l'utilisent.

> Repère les motions encore faibles / peu représentatives et signale-les — méthode : rejouer
> nom → symbole → mouvement (cf `TASKS.md`, passes 1→5). Source : `tools/motions.py`.
