# WLED Contrôles

[Palettes](palettes.fr.md) · [Effets](effects.fr.md) · **Contrôles**

<sub>Autres langues: [EN](controls.en.md) · [DE](controls.de.md) · [ES](controls.es.md) · [IT](controls.it.md) · [JA](controls.ja.md) · [KO](controls.ko.md) · [ZH](controls.zh.md)</sub>

| Nom WLED | Traduction | Description |
|---|---|---|
| `Speed` | Vitesse | À quelle vitesse l'effet s'anime. |
| `Intensity` | Intensité | L'intensité ou la densité de l'effet. |
| `Custom 1` | Perso 1 | Curseur propre à l'effet (sens variable). |
| `Custom 2` | Perso 2 | Deuxième curseur propre à l'effet. |
| `Custom 3` | Perso 3 | Troisième curseur de l'effet (0-31). |
| `Option 1` | Option 1 | Bascule oui/non propre à l'effet. |
| `Option 2` | Option 2 | Deuxième bascule de l'effet. |
| `Option 3` | Option 3 | Troisième bascule de l'effet. |
| `Effect color` | Couleur d'effet | La couleur principale de l'effet. |
| `Background color` | Couleur de fond | La couleur de fond de l'effet. |
| `Custom color` | Couleur perso | Troisième emplacement de couleur. |
| `Palette` | Palette | La palette de couleurs utilisée par l'effet. |
