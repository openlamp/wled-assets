Les **champs Info** sont les valeurs du panneau **Info** de WLED (`/json/info`) — temps de fonctionnement, mémoire libre, signal, FPS, courant estimé, etc.

| Image | Nom WLED | Traduction | Description |
|---|---|---|---|
| <img src="../../images/info/uptime.png" width="56"> | `Uptime` | Temps de fonctionnement | How long the device has been running since boot. |
| <img src="../../images/info/free-memory.png" width="56"> | `Free memory` | Mémoire libre | Free heap (RAM) on the controller. |
| <img src="../../images/info/signal-strength.png" width="56"> | `Signal strength` | Force du signal | Wi-Fi signal quality. |
| <img src="../../images/info/frames-per-second.png" width="56"> | `Frames per second` | Images par seconde | Effect refresh rate (FPS). |
| <img src="../../images/info/estimated-current.png" width="56"> | `Estimated current` | Courant estimé | Estimated power draw of the LEDs. |
| <img src="../../images/info/led-count.png" width="56"> | `LED count` | Nombre de LED | Total LEDs configured. |
| <img src="../../images/info/environment.png" width="56"> | `Environment` | Environnement | Build/platform info (ESP variant, WLED version). |
| <img src="../../images/info/filesystem.png" width="56"> | `Filesystem` | Système de fichiers | On-device storage usage. |
| <img src="../../images/info/mac-address.png" width="56"> | `MAC address` | Adresse MAC | Hardware network address. |
| <img src="../../images/info/ip-address.png" width="56"> | `IP address` | Adresse IP | The device's network address. |
| <img src="../../images/info/wifi-channel.png" width="56"> | `WiFi channel` | Canal Wi-Fi | The Wi-Fi channel in use. |
| <img src="../../images/info/version.png" width="56"> | `Version` | Version | WLED firmware version. |
