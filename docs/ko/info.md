| 이미지 | WLED 이름 | 번역 | 설명 |
|---|---|---|---|
| <img src="../../images/info/uptime.png" width="56"> | `Uptime` | 가동 시간 | How long the device has been running since boot. |
| <img src="../../images/info/free-memory.png" width="56"> | `Free memory` | 여유 메모리 | Free heap (RAM) on the controller. |
| <img src="../../images/info/signal-strength.png" width="56"> | `Signal strength` | 신호 세기 | Wi-Fi signal quality. |
| <img src="../../images/info/frames-per-second.png" width="56"> | `Frames per second` | 초당 프레임 | Effect refresh rate (FPS). |
| <img src="../../images/info/estimated-current.png" width="56"> | `Estimated current` | 예상 전류 | Estimated power draw of the LEDs. |
| <img src="../../images/info/led-count.png" width="56"> | `LED count` | LED 수 | Total LEDs configured. |
| <img src="../../images/info/environment.png" width="56"> | `Environment` | 환경 | Build/platform info (ESP variant, WLED version). |
| <img src="../../images/info/filesystem.png" width="56"> | `Filesystem` | 파일 시스템 | On-device storage usage. |
| <img src="../../images/info/mac-address.png" width="56"> | `MAC address` | MAC 주소 | Hardware network address. |
| <img src="../../images/info/ip-address.png" width="56"> | `IP address` | IP 주소 | The device's network address. |
| <img src="../../images/info/wifi-channel.png" width="56"> | `WiFi channel` | Wi-Fi 채널 | The Wi-Fi channel in use. |
| <img src="../../images/info/version.png" width="56"> | `Version` | 버전 | WLED firmware version. |
