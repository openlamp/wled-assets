**세그먼트 동작**은 **세그먼트** 탭의 세그먼트별 옵션(반전, 미러, 정지, 켜기/끄기, 전환, 오디오 시뮬). `seg.rev/mi/frz/on` 등.

| 이미지 | WLED 이름 | 번역 | 설명 |
|---|---|---|---|
| <img src="../../images/segment/reverse.png" width="56"> | `Reverse` | 반전 | 효과를 세그먼트에서 반대 방향으로 재생. |
| <img src="../../images/segment/mirror.png" width="56"> | `Mirror` | 미러 | 세그먼트를 나눠 중앙에서 대칭. |
| <img src="../../images/segment/freeze.png" width="56"> | `Freeze` | 정지 | 현재 프레임에서 효과 정지. |
| <img src="../../images/segment/on-off.png" width="56"> | `On/Off` | 켜기/끄기 | 세그먼트 켜기/끄기. |
| <img src="../../images/segment/transition.png" width="56"> | `Transition` | 전환 | 상태/프리셋 전환 시 크로스페이드 시간. |
| <img src="../../images/segment/sound-sim.png" width="56"> | `Sound sim` | 오디오 시뮬 | 실시간 오디오가 없을 때 오디오 시뮬레이션. |
