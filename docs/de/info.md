**Info-Felder** sind die Werte im **Info**-Panel von WLED (`/json/info`) — Betriebszeit, freier Speicher, Signal, FPS, geschätzter Strom usw.

| Bild | WLED-Name | Übersetzung | Beschreibung |
|---|---|---|---|
| <img src="../../images/info/uptime.png" width="56"> | `Uptime` | Betriebszeit | How long the device has been running since boot. |
| <img src="../../images/info/free-memory.png" width="56"> | `Free memory` | Freier Speicher | Free heap (RAM) on the controller. |
| <img src="../../images/info/signal-strength.png" width="56"> | `Signal strength` | Signalstärke | Wi-Fi signal quality. |
| <img src="../../images/info/frames-per-second.png" width="56"> | `Frames per second` | Bilder pro Sekunde | Effect refresh rate (FPS). |
| <img src="../../images/info/estimated-current.png" width="56"> | `Estimated current` | Geschätzter Strom | Estimated power draw of the LEDs. |
| <img src="../../images/info/led-count.png" width="56"> | `LED count` | LED-Anzahl | Total LEDs configured. |
| <img src="../../images/info/environment.png" width="56"> | `Environment` | Umgebung | Build/platform info (ESP variant, WLED version). |
| <img src="../../images/info/filesystem.png" width="56"> | `Filesystem` | Dateisystem | On-device storage usage. |
| <img src="../../images/info/mac-address.png" width="56"> | `MAC address` | MAC-Adresse | Hardware network address. |
| <img src="../../images/info/ip-address.png" width="56"> | `IP address` | IP-Adresse | The device's network address. |
| <img src="../../images/info/wifi-channel.png" width="56"> | `WiFi channel` | WLAN-Kanal | The Wi-Fi channel in use. |
| <img src="../../images/info/version.png" width="56"> | `Version` | Version | WLED firmware version. |
