| Nombre WLED | Traducción | Descripción |
|---|---|---|
| `Speed` | Velocidad | La rapidez de la animación del efecto. |
| `Intensity` | Intensidad | La intensidad o densidad del efecto. |
| `Custom 1` | Personal. 1 | Deslizador propio del efecto (varía). |
| `Custom 2` | Personal. 2 | Segundo deslizador del efecto. |
| `Custom 3` | Personal. 3 | Tercer deslizador del efecto (0-31). |
| `Option 1` | Opción 1 | Interruptor propio del efecto. |
| `Option 2` | Opción 2 | Segundo interruptor del efecto. |
| `Option 3` | Opción 3 | Tercer interruptor del efecto. |
| `Effect color` | Color de efecto | El color principal del efecto. |
| `Background color` | Color de fondo | El color de fondo del efecto. |
| `Custom color` | Color personal. | Tercer espacio de color. |
| `Palette` | Paleta | La paleta de colores que usa el efecto. |
