Los **campos de info** son los datos del panel **Info** de WLED (`/json/info`) — tiempo activo, memoria libre, señal, FPS, corriente estimada, etc.

| Imagen | Nombre WLED | Traducción | Descripción |
|---|---|---|---|
| <img src="../../images/info/uptime.png" width="56"> | `Uptime` | Tiempo activo | How long the device has been running since boot. |
| <img src="../../images/info/free-memory.png" width="56"> | `Free memory` | Memoria libre | Free heap (RAM) on the controller. |
| <img src="../../images/info/signal-strength.png" width="56"> | `Signal strength` | Intensidad de señal | Wi-Fi signal quality. |
| <img src="../../images/info/frames-per-second.png" width="56"> | `Frames per second` | Fotogramas por segundo | Effect refresh rate (FPS). |
| <img src="../../images/info/estimated-current.png" width="56"> | `Estimated current` | Corriente estimada | Estimated power draw of the LEDs. |
| <img src="../../images/info/led-count.png" width="56"> | `LED count` | Número de LED | Total LEDs configured. |
| <img src="../../images/info/environment.png" width="56"> | `Environment` | Entorno | Build/platform info (ESP variant, WLED version). |
| <img src="../../images/info/filesystem.png" width="56"> | `Filesystem` | Sistema de archivos | On-device storage usage. |
| <img src="../../images/info/mac-address.png" width="56"> | `MAC address` | Dirección MAC | Hardware network address. |
| <img src="../../images/info/ip-address.png" width="56"> | `IP address` | Dirección IP | The device's network address. |
| <img src="../../images/info/wifi-channel.png" width="56"> | `WiFi channel` | Canal Wi-Fi | The Wi-Fi channel in use. |
| <img src="../../images/info/version.png" width="56"> | `Version` | Versión | WLED firmware version. |
