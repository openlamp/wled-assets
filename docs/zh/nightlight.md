| WLED 名称 | 翻译 | 描述 |
|---|---|---|
| `Instant` | 即时 | 计时结束时立即熄灭 — 无渐变。 |
| `Fade` | 渐暗 | 在整个时长内亮度逐渐降低。 |
| `Colour fade` | 颜色渐变 | 亮度和颜色一起渐变到夜灯颜色。 |
| `Sunrise` | 日出 | 唤醒灯：亮度在时长内逐渐升高，如日出。 |
