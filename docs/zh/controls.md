| WLED 名称 | 翻译 | 描述 |
|---|---|---|
| `Speed` | 速度 | 效果动画的快慢。 |
| `Intensity` | 强度 | 效果的强度或密度。 |
| `Custom 1` | 自定义 1 | 效果专属滑块（含义随效果而变）。 |
| `Custom 2` | 自定义 2 | 第二个效果专属滑块。 |
| `Custom 3` | 自定义 3 | 第三个滑块（0-31）。 |
| `Option 1` | 选项 1 | 效果专属开关。 |
| `Option 2` | 选项 2 | 第二个开关。 |
| `Option 3` | 选项 3 | 第三个开关。 |
| `Effect color` | 效果颜色 | 效果的主色。 |
| `Background color` | 背景色 | 效果的背景颜色。 |
| `Custom color` | 自定义颜色 | 第三个颜色槽。 |
| `Palette` | 调色板 | 效果所用的配色板。 |
