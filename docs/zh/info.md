| WLED 名称 | 翻译 | 描述 |
|---|---|---|
| `Uptime` | 运行时间 | How long the device has been running since boot. |
| `Free memory` | 可用内存 | Free heap (RAM) on the controller. |
| `Signal strength` | 信号强度 | Wi-Fi signal quality. |
| `Frames per second` | 每秒帧数 | Effect refresh rate (FPS). |
| `Estimated current` | 估计电流 | Estimated power draw of the LEDs. |
| `LED count` | LED 数量 | Total LEDs configured. |
| `Environment` | 环境 | Build/platform info (ESP variant, WLED version). |
| `Filesystem` | 文件系统 | On-device storage usage. |
| `MAC address` | MAC 地址 | Hardware network address. |
| `IP address` | IP 地址 | The device's network address. |
| `WiFi channel` | Wi-Fi 信道 | The Wi-Fi channel in use. |
| `Version` | 版本 | WLED firmware version. |
