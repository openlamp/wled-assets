**信息字段**是 WLED **信息**面板（`/json/info`）中的读数——运行时间、可用内存、信号、FPS、估计电流等。

| 图片 | WLED 名称 | 翻译 | 描述 |
|---|---|---|---|
| <img src="../../images/info/uptime.png" width="56"> | `Uptime` | 运行时间 | How long the device has been running since boot. |
| <img src="../../images/info/free-memory.png" width="56"> | `Free memory` | 可用内存 | Free heap (RAM) on the controller. |
| <img src="../../images/info/signal-strength.png" width="56"> | `Signal strength` | 信号强度 | Wi-Fi signal quality. |
| <img src="../../images/info/frames-per-second.png" width="56"> | `Frames per second` | 每秒帧数 | Effect refresh rate (FPS). |
| <img src="../../images/info/estimated-current.png" width="56"> | `Estimated current` | 估计电流 | Estimated power draw of the LEDs. |
| <img src="../../images/info/led-count.png" width="56"> | `LED count` | LED 数量 | Total LEDs configured. |
| <img src="../../images/info/environment.png" width="56"> | `Environment` | 环境 | Build/platform info (ESP variant, WLED version). |
| <img src="../../images/info/filesystem.png" width="56"> | `Filesystem` | 文件系统 | On-device storage usage. |
| <img src="../../images/info/mac-address.png" width="56"> | `MAC address` | MAC 地址 | Hardware network address. |
| <img src="../../images/info/ip-address.png" width="56"> | `IP address` | IP 地址 | The device's network address. |
| <img src="../../images/info/wifi-channel.png" width="56"> | `WiFi channel` | Wi-Fi 信道 | The Wi-Fi channel in use. |
| <img src="../../images/info/version.png" width="56"> | `Version` | 版本 | WLED firmware version. |
