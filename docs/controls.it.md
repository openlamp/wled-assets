| Nome WLED | Traduzione | Descrizione |
|---|---|---|
| `Speed` | Velocità | Quanto velocemente si anima l'effetto. |
| `Intensity` | Intensità | L'intensità o densità dell'effetto. |
| `Custom 1` | Custom 1 | Cursore specifico dell'effetto (varia). |
| `Custom 2` | Custom 2 | Secondo cursore dell'effetto. |
| `Custom 3` | Custom 3 | Terzo cursore dell'effetto (0-31). |
| `Option 1` | Opzione 1 | Interruttore specifico dell'effetto. |
| `Option 2` | Opzione 2 | Secondo interruttore dell'effetto. |
| `Option 3` | Opzione 3 | Terzo interruttore dell'effetto. |
| `Effect color` | Colore effetto | Il colore principale dell'effetto. |
| `Background color` | Colore di sfondo | Il colore di sfondo dell'effetto. |
| `Custom color` | Colore custom | Terzo slot di colore. |
| `Palette` | Tavolozza | La tavolozza usata dall'effetto. |
