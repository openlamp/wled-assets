| WLED名 | 翻訳 | 説明 |
|---|---|---|
| `Reverse` | 反転 | 効果をセグメント上で逆方向に再生。 |
| `Mirror` | ミラー | セグメントを分割し中央から鏡像化。 |
| `Freeze` | フリーズ | 現在のフレームで効果を停止。 |
| `On/Off` | オン/オフ | セグメントのオン/オフ。 |
| `Transition` | トランジション | 状態/プリセット切替時のクロスフェード時間。 |
| `Sound sim` | 音声シミュ | ライブ音声がない時の音声シミュレーション。 |
