| WLED名 | 翻訳 | 説明 |
|---|---|---|
| `Speed` | 速度 | エフェクトのアニメーション速度。 |
| `Intensity` | 強度 | エフェクトの強さ・密度。 |
| `Custom 1` | カスタム1 | エフェクト固有のスライダー(意味は効果次第)。 |
| `Custom 2` | カスタム2 | エフェクト固有の2つ目のスライダー。 |
| `Custom 3` | カスタム3 | 3つ目のスライダー(0-31)。 |
| `Option 1` | オプション1 | エフェクト固有のオン/オフ。 |
| `Option 2` | オプション2 | 2つ目のオン/オフ。 |
| `Option 3` | オプション3 | 3つ目のオン/オフ。 |
| `Effect color` | エフェクト色 | エフェクトの主色。 |
| `Background color` | 背景色 | エフェクトの背景色。 |
| `Custom color` | カスタム色 | 3つ目の色スロット。 |
| `Palette` | パレット | エフェクトが使う配色パレット。 |
