**Info fields** are the read-outs in WLED's **Info** panel (`/json/info`) — uptime, free memory, signal, FPS, estimated current and so on.

| Image | WLED name | Translation | Description |
|---|---|---|---|
| <img src="../../images/info/uptime.png" width="56"> | `Uptime` | Uptime | How long the device has been running since boot. |
| <img src="../../images/info/free-memory.png" width="56"> | `Free memory` | Free memory | Free heap (RAM) on the controller. |
| <img src="../../images/info/signal-strength.png" width="56"> | `Signal strength` | Signal strength | Wi-Fi signal quality. |
| <img src="../../images/info/frames-per-second.png" width="56"> | `Frames per second` | Frames per second | Effect refresh rate (FPS). |
| <img src="../../images/info/estimated-current.png" width="56"> | `Estimated current` | Estimated current | Estimated power draw of the LEDs. |
| <img src="../../images/info/led-count.png" width="56"> | `LED count` | LED count | Total LEDs configured. |
| <img src="../../images/info/environment.png" width="56"> | `Environment` | Environment | Build/platform info (ESP variant, WLED version). |
| <img src="../../images/info/filesystem.png" width="56"> | `Filesystem` | Filesystem | On-device storage usage. |
| <img src="../../images/info/mac-address.png" width="56"> | `MAC address` | MAC address | Hardware network address. |
| <img src="../../images/info/ip-address.png" width="56"> | `IP address` | IP address | The device's network address. |
| <img src="../../images/info/wifi-channel.png" width="56"> | `WiFi channel` | WiFi channel | The Wi-Fi channel in use. |
| <img src="../../images/info/version.png" width="56"> | `Version` | Version | WLED firmware version. |
