| WLED name | Translation | Description |
|---|---|---|
| `Instant` | Instant | The light switches off at once when the timer ends — no fade. |
| `Fade` | Fade | Brightness fades down gradually over the whole duration. |
| `Colour fade` | Colour fade | Fades brightness AND colour toward the nightlight colour. |
| `Sunrise` | Sunrise | Wake-up light: brightness fades UP over the duration, like a sunrise. |
