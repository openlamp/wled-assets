|---|---|---|
| `Uptime` | Tempo di attività | How long the device has been running since boot. |

| `Signal strength` | Intensità del segnale | Wi-Fi signal quality. |

| `Environment` | Ambiente | Build/platform info (ESP variant, WLED version). |

| `MAC address` | Indirizzo MAC | Hardware network address. |
| `IP address` | Indirizzo IP | The device's network address. |
| `WiFi channel` | Canale Wi-Fi | The Wi-Fi channel in use. |
