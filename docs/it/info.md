I **campi info** sono i valori nel pannello **Info** di WLED (`/json/info`) — tempo di attività, memoria libera, segnale, FPS, corrente stimata, ecc.

| Immagine | Nome WLED | Traduzione | Descrizione |
|---|---|---|---|
| <img src="../../images/info/uptime.png" width="56"> | `Uptime` | Tempo di attività | How long the device has been running since boot. |
| <img src="../../images/info/free-memory.png" width="56"> | `Free memory` | Memoria libera | Free heap (RAM) on the controller. |
| <img src="../../images/info/signal-strength.png" width="56"> | `Signal strength` | Intensità del segnale | Wi-Fi signal quality. |
| <img src="../../images/info/frames-per-second.png" width="56"> | `Frames per second` | Fotogrammi al secondo | Effect refresh rate (FPS). |
| <img src="../../images/info/estimated-current.png" width="56"> | `Estimated current` | Corrente stimata | Estimated power draw of the LEDs. |
| <img src="../../images/info/led-count.png" width="56"> | `LED count` | Numero di LED | Total LEDs configured. |
| <img src="../../images/info/environment.png" width="56"> | `Environment` | Ambiente | Build/platform info (ESP variant, WLED version). |
| <img src="../../images/info/filesystem.png" width="56"> | `Filesystem` | File system | On-device storage usage. |
| <img src="../../images/info/mac-address.png" width="56"> | `MAC address` | Indirizzo MAC | Hardware network address. |
| <img src="../../images/info/ip-address.png" width="56"> | `IP address` | Indirizzo IP | The device's network address. |
| <img src="../../images/info/wifi-channel.png" width="56"> | `WiFi channel` | Canale Wi-Fi | The Wi-Fi channel in use. |
| <img src="../../images/info/version.png" width="56"> | `Version` | Versione | WLED firmware version. |
