| WLED name | Translation | Description |
|---|---|---|
| `Speed` | Speed | How fast the effect animates. |
| `Intensity` | Intensity | The effect's intensity or density. |
| `Custom 1` | Custom 1 | An effect-specific slider (meaning varies by effect). |
| `Custom 2` | Custom 2 | A second effect-specific slider. |
| `Custom 3` | Custom 3 | A third effect-specific slider (0-31). |
| `Option 1` | Option 1 | An effect-specific on/off toggle. |
| `Option 2` | Option 2 | A second effect-specific toggle. |
| `Option 3` | Option 3 | A third effect-specific toggle. |
| `Effect color` | Effect color | The effect's primary colour. |
| `Background color` | Background color | The effect's background colour. |
| `Custom color` | Custom color | A third colour slot for the effect. |
| `Palette` | Palette | The colour palette the effect draws from. |
